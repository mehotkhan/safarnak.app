سفرناک چیست؟

سفرناک یک همراه هوشمند آفلاین برای برنامه‌ریزی، مدیریت و تجربه‌ی بهتر سفرهای شماست.

چرا سفرناک؟

مزیت‌هایی که سفرناک را متفاوت می‌کند

  • • کاملاً آفلاین: بدون اینترنت هم کار می‌کند
  • • همگام‌سازی خودکار: پس از اتصال، همه چیز به‌روز می‌شود
  • • شخصی‌سازی: برنامه سفر مطابق سلیقه و سبک سفر شما
  • • پشتیبانی از زبان فارسی و راست‌به‌چپ
  • • API قدرتمند GraphQL برای توسعه‌دهندگان
  • • امنیت: هش رمز عبور PBKDF2 و معماری امن سمت سرور
  • • ساختار کد تمیز و مستند برای مشارکت آسان
  • • طراحی مدرن با Tailwind و حالت تیره/روشن

مستندات کامل GraphQL

راهنمای توسعه‌دهندگان

چگونه یک قابلیت جدید اضافه کنیم؟

مراحل کلی

  1. تعریف در graphql/schema.graphql
  2. ایجاد فایل عملیات در graphql/queries
  3. اجرای yarn codegen برای تولید هوک‌ها
  4. پیاده‌سازی ریزالور در worker/queries یا worker/mutations
  5. استفاده از هوک‌ها در اپلیکیشن با @api

شروع سریع توسعه

# نصب و راه‌اندازی
 git clone https://github.com/mehotkhan/safarnak.app.git
 cd safarnak.app
 yarn install
 yarn db:migrate
 yarn codegen
 yarn dev   # اجرای همزمان Worker و Expo
